# About the project 

- This is a VueJS pure project. My personal website and blog. 
- It uses yarn, TailwindCSS and DaisyUI
- It is a pure SPA application
- The main code is located under src/
- The blog consists of a series of Markdown articles under blog/
- We use Yarn in this application. 
  - You should try to run the commands under package.json before any NPM extra commands
  - To test specific Server Side code build, use vite-ssg
- You are not required to build the system every time. Whenever required, I'll ask you.
  - When you try to build to system, try to fix it only once.
  - If you are not able to fix automatically, summarize the error and report.
