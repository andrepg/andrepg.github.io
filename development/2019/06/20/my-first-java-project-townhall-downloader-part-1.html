My first Java project: a Downloader and PDF converter - #1

Simple briefing about this app

We provide some Invoices to our customers. These documents has to be saved into a folder in a PDF format. As our number of documents increase, the work needed increases. When I’m writing this post, we generate almost 100 documents.

So, It’s a job of search, view and print each document with a PDF printer. A hundred times.

Then I tought: Why not develop a tool to do this job? So, I started to search documentation and how to do this program. I’ve discovered some proprietary tools from TownHall and could make these app work.

It’s based on a HTML render tool and PDF generator

I’ve discovered a XML file with the entire information about all documents generated in a month. The TownHall provides this file into a reserved section, to authorized only. With this file, I was able to read the number and verification code of document. This allow me to esarch a specific URL and get the HTML representation.

It’s simple: we get a real URL, make a request, render with a hidden HTML renderer and save it into a PDF file with some thirdy party library.
